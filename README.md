# Employee Database Management System

A desktop application built using Python, Tkinter and MySQL to manage employee records.

## Features
- User Login
- Add, Update, Delete Employee
- Assign Employee to Projects
- Salary Statistics
- Filter Employees by Name & Department

## Tech Stack
- Python
- Tkinter
- MySQL

## How to Run
1. Run database.sql in MySQL
2. Install requirements  
   pip install -r requirements.txt
3. Run the app  
   python employee_app.py
